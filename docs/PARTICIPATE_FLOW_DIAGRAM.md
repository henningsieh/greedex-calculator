## Form Validation Rules

```
Step 1 (Welcome):
  - name: required, non-empty after trim
  - country: required, non-empty after trim

Step 2 (Transport):
  - transports array: minimum 1 entry required
  - each transport:
    - type: must be valid ActivityType
    - distanceKm: must be > 0

Step 3 (Accommodation):
  - type: required, from valid set
  - nights: required, integer > 0
  - energyType: required, "conventional" or "green"

Step 4 (Review):
  - All previous validations must pass
  - Review only, no new validation
```
